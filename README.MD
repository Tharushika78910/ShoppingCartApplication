## How to Use the Application

1. Open the application.

2. Select a language from the **Select Language** dropdown.

3. Click **Confirm Language** to apply the selected language.
    - The application title and UI texts will update based on the selected language.

4. Enter the number of items you want to add to the shopping cart in the **Number of Items** field.

5. Click **Enter Items**.
    - Input fields for **price** and **quantity** will appear for each item.

6. For each item, enter:
    - **Price**
    - **Quantity**

7. Click **Calculate Total**.
    - The application will calculate:
        - each item subtotal
        - the total cart cost

8. After calculation, the application automatically saves:
    - the cart summary into the `cart_records` table
    - the individual items into the `cart_items` table

9. To test another language or another cart, change the language or enter a new number of items and repeat the steps.
